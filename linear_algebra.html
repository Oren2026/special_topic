Ch1 線性方程式系統 ｜Linear Systems of Equations

面對多個未知數、多條約束時，如何系統化地找到解——或者判斷根本無解。兩種核心算法：高斯消去法（得到解）和高斯-喬登消去法（得到完整形式）。

線性方程式系統簡介 高斯消去法 & 高斯-喬登消去法 線性方程式系統的應用
§1.1 線性方程式系統簡介

什麼是「線性」方程式？

「線性」的意思是：未知數只會以「一次方」出現（沒有 x²、xy、sin(x) 這類東西），而且未知數之間不會相乘。

Linear（線性）的定義 每個方程式都是未知數的「線性組合 = 常數」的形式。
a₁x₁ + a₂x₂ + ... + aₙxₙ = b
其中 aᵢ 和 b 是已知常數，xᵢ 是未知數。

這看起來很限制，但它其實是大部分科學與工程問題的第一步近似——先搞清楚「線性的部分」，非線性的部分再慢慢處理。

方程組的標準形式

把 m 條方程式、n 個未知數寫在一起：

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁ a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂ ⋮ ⋮ aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ

這就是「線性方程式系統」。有 m 條方程式，n 個未知數。我們的目標：找到所有滿足這些方程式的 (x₁, x₂, ..., xₙ)。

方程組的三種解的情況

情況一：唯一解
有且僅有一組 (x₁, x₂, ..., xₙ) 滿足所有方程式
條件：rank(A) = rank([A|b]) = n
情況二：無限多解
有无限多組解，解形成一個集合
條件：rank(A) = rank([A|b]) < n，有自由變數
情況三：無解
沒有任何解同時滿足所有方程式
條件：rank(A) < rank([A|b])，矛盾的行

如何快速判斷解的情況

把增廣矩陣做高斯消去（只化到階梯形），觀察：

  • 化簡後，出現形如 [0 0 ... 0 | c]（c ≠ 0）的行 → 無解
  • 沒有矛盾行，且自由變數（free variable）個數 > 0 → 無限多解
  • 沒有矛盾行，且每個變數都是主元變數（pivot variable）→ 唯一解
§1.2 高斯消去法與高斯-喬登消去法

高斯消去法 ｜Gaussian Elimination

Forward Elimination + Back Substitution

將增廣矩陣 [A|b] 通過列運算化為上三角形（Row Echelon Form），然後從最後一行往前回代求出所有未知數。

⚙️ 標準步驟

  1. 寫出增廣矩陣 [A|b] — 把係數和常數寫在同一個矩陣
  2. 選第一列第一個非零元素為主元（pivot）
  3. 用列交換或列運算，把 pivot 下面的元素變成 0（前向消去）
  4. 對第二列、第三列... 重複，直到矩陣變成上三角形
  5. 從最後一行開始回代（Back Substitution），逐一求出每個未知數
pivot列交換列運算前向消去回代REF
💡 複雜度：O(n³) — 所有矩陣運算的基礎複雜度

高斯-喬登消去法 ｜Gauss-Jordan Elimination

完全化簡 → 直接讀出解

在高斯消去之後，繼續把每列的主元上方也消成 0，直到矩陣變成「簡化列階梯形」（RREF）。這時可以直接讀出解——不需要回代。

⚙️ 與高斯消去的差異

  1. 做完高斯消去（上三角形）之後，不停下
  2. 繼續往上消——把每個 pivot 上方的元素也變成 0
  3. 最後結果：每個 pivot 是它所在列唯一的非零元素，且等於 1
  4. 好處：直接讀出 x 的值，不需要回代
RREFpivot = 1簡化列階梯形不需要回代
💡 適合：用於求逆矩陣、解方程組、確認向量是否為解

高斯 vs 高斯-喬登 — 什麼時候用哪個？

方法做到什麼程度計算量適用場景
高斯消去 上三角形（REF） 較少 只求方程組的解
高斯-喬登 簡化列階梯形（RREF） 較多（多約50%） 求逆矩陣、確認解的狀態、多個右邊常數

基本列運算（允許的三種操作）

↔️ 交換（Swap）

Ri ↔ Rj

當 pivot 位置是 0 時使用。對應置換矩陣 P。

× 倍加（Scale）

Ri → c·Ri（c ≠ 0）

把一列所有元素乘以非零常數。

+ 替換（Replace）

Ri → Ri + c·Rj

最核心的操作。用於把元素消成 0。

💡 核心洞察 這三種操作不改變方程組的解集合。它們只是把方程組「換個樣子表示」，本質不變。
§1.3 線性方程式系統的應用

⚡ 電網節點分析（Node Analysis）

KCL：克希荷夫電流定律

流入節點的電流總和 = 流出節點的電流總和。N 個節點 → N-1 條線性方程式。

Σ Iᵢ = 0 (at each node)
💬 比喻 就像自來水管網的節點——進入的水量必須等於離開的水量。

📊 化學反應平衡

質量守恆定律

化學反應的反應物和生成物必須滿足原子守恆。配平化學方程式 = 解線性方程組。

💬 比喻 就像在餐廳算帳——所有食材進來，必須等於所有菜品出去的質量。

🌐 網路流量規劃

流量守恆

交通網絡、電網、互聯網路由器——每個節點的流入流量等於流出流量。

💬 比喻 就像城市交通：進入每個路口的車流量，必須等於離開的車流量。

📈 供需均衡模型

經濟學中的線性系統

N 種商品的市場供需均衡：每種商品的供給 = 需求，形成 N 條方程式。

💬 比喻 就像拍賣市場——當買家願付的價格等於賣家願收的價格時，市場成交。

Ch2 矩陣 ｜Matrix

矩陣不只是「記錄方程組」的容器——它本身是一個獨立的數學物件，有自己的代數規則。這章把「數字方塊」變成可以加減乘除的代數系統。

§2.1 矩陣運算

矩陣是什麼

一個 m×n 的矩陣是一個 m 列、n 行的數字長方形陣列。寫成 A(m×n) 表示 m 列、n 行。

A = | a₁₁ a₁₂ ... a₁ₙ | | a₂₁ a₂₂ ... a₂ₙ | | ⋮ ⋮ ⋱ ⋮ | | aₘ₁ aₘ₂ ... aₘₙ |

把線性方程組寫成矩陣形式：

A(m×n) · x(n×1) = b(m×1)

A 是係數矩陣，x 是未知數向量，b 是常數向量。

💡 幾何意涵 A(m×n) 是一個從 Rⁿ 空間到 Rᵐ 空間的「線性變換」。它把 n 維輸入向量 transform 成 m 維輸出向量。

矩陣的四種基本運算

➕ 矩陣加法

同維度的兩個矩陣，每個位置各自相加。

| 1 2 | | 5 6 | | 6 8 | | 3 4 | + | 7 8 | = | 10 12 |

條件：A 和 B 必須同維度

✖️ 純量乘法

每個元素都乘以該純量 c。

3 × | 1 2 | = | 3 6 | | 3 4 | | 9 12 |

✖️ 矩陣乘法

A(m×n) · B(n×p) = C(m×p)

C 的第 (i,j) 元素 = A 的第 i 行點乘 B 的第 j 列

ᵀ 轉置（Transpose）

行列交換。第 (i,j) 元素變成第 (j,i) 元素。

| 1 2 3 |ᵀ | 1 4 | | 4 5 6 | = | 2 5 | | 3 6 |

矩陣乘法的關鍵性質

⚠️ 矩陣乘法 ≠ 普通數字乘法
  • 交換律一般不成立： AB ≠ BA（通常不相等，甚至維度都不允許比較）
  • 零因子問題： AB = 0 不意味著 A=0 或 B=0
  • 消去律不成立： AB = AC 不意味著 B = C
✅ 成立的性質
  • 結合律：(AB)C = A(BC)
  • 左分配律：A(B+C) = AB + AC
  • 右分配律：(A+B)C = AC + BC

矩陣乘法的維度規則

A (m×n) · B (n×p) = C (m×p)

記住：中間維度（n）必須匹配，結果維度是「外側」的 m×p。

💡 記憶技巧 把維度豎著寫：A(m×**n**) · B(**n**×p) = C(m×p)。「吃進去的中間，吐出來的外側」。
§2.2 矩陣運算的性質

特殊矩陣類型

I 單位矩陣 I

對角線為1，其他位置為0。n×n。

I₃ = | 1 0 0 | | 0 1 0 | | 0 0 1 |

A·I = I·A = A（矩陣中的「1」）

D 對角矩陣

只在對角線有元素，其他為0。

D = diag(d₁, d₂, ..., dₙ)

D·x = [d₁x₁, d₂x₂, ..., dₙxₙ] — 乘法很快

↗️ 上/下三角矩陣

上三角：對角線以下全是0。
下三角：對角線以上全是0。

上三角 = | ★ ★ ★ | | 0 ★ ★ | | 0 0 ★ |

det(三角矩陣) = 對角線元素乘積

S 對稱矩陣

A = Aᵀ。關於主對角線對稱。

| 1 4 5 | | 4 2 6 | | 5 6 3 |

真實世界的很多矩陣是對稱的（距離矩陣、共變異數矩陣）

P 置換矩陣

每行、每列恰好只有一個1，其餘為0。

| 0 1 0 | | 1 0 0 | | 0 0 1 |

用於「記錄列交換」：PA = LU 中的 P

O 零矩陣

所有元素為0。

O + A = A · O·A = O · 但要注意：O·A = O 不意味著 O = 0 或 A = 0！

矩陣的冪次

A² = A·A，A³ = A·A·A。當 A 是方陣（n×n）時才有意義。

💡 實際應用 在馬可夫鏈中，Aᵏ 告訴你從狀態 i 經過 k 步轉移到狀態 j 的機率。
§2.3 反矩陣（Inverse Matrix）

什麼是反矩陣

對於方陣 A(n×n)，如果存在矩陣 B 使得 AB = BA = Iₙ，則 B 稱為 A 的反矩陣，記為 A⁻¹。

A · A⁻¹ = I 且 A⁻¹ · A = I
可逆（Invertible）
det(A) ≠ 0
又稱：nonsingular · nondegenerate
不可逆（Singular）
det(A) = 0
又稱：noninvertible · degenerate

如何求反矩陣 — 高斯-喬登法

Augmented matrix trick

把 [A | I] 放在一起，用高斯-喬登消去把左邊變成 I，右邊剩下的就是 A⁻¹。

⚙️ 步驟

  1. 寫出增廣矩陣 [A | Iₙ]（左邊 A，右邊同維度單位矩陣）
  2. 對這個組合矩陣做高斯-喬登消去，把左邊變成 RREF
  3. 如果左邊變成 I，右邊就是 A⁻¹
  4. 如果左邊出現零列（rank不足）→ A 不可逆
RREF[A|I]nonsingulardet(A) ≠ 0

反矩陣的重要性質

性質公式
反矩陣的反矩陣(A⁻¹)⁻¹ = A
乘積的反矩陣(AB)⁻¹ = B⁻¹A⁻¹（順序反過來！）
純量乘法(cA)⁻¹ = (1/c)·A⁻¹ (c ≠ 0)
轉置的反矩陣(Aᵀ)⁻¹ = (A⁻¹)ᵀ

反矩陣的應用：解線性方程組

當 A 可逆時，Ax = b 的唯一解可以直接寫成：

x = A⁻¹ · b
⚠️ 實際計算注意 直接算 A⁻¹ 需要 O(n³)，用高斯消去解 Ax = b 也需要 O(n³)，但常數更小。除非你要解很多個同樣的 A 但不同的 b，否則一般不推薦先算反矩陣再乘。
§2.4 基本矩陣（Elementary Matrices）

什麼是基本矩陣

單位矩陣 I 做一次「基本列運算」（交換、倍加、替換）之後得到的矩陣。

E · A = 對 A 進行對應列運算後的結果

也就是說：對 A 做一次列運算 = 在左邊乘上一個基本矩陣 E。

↔️ 列交換型

I 的第 i 列和第 j 列交換後的結果。反矩陣 = 自身（交換兩次回到原樣）。

× 列倍加型

I 的第 i 列乘以純量 c (c ≠ 0)。反矩陣：第 i 列乘以 1/c。

+ 列替換型

I 的第 j 列加上第 i 列的 c 倍。反矩陣：第 j 列減去第 i 列的 c 倍。

為什麼基本矩陣重要 — LU 分解的原理

高斯消去把 A 變成 U（上三角）的過程，相當於一系列基本矩陣的乘法：

Eₖ · ... · E₂ · E₁ · A = U

因此：

A = E₁⁻¹ · E₂⁻¹ · ... · Eₖ⁻¹ · U = L · U

L 是下三角矩陣（因為基本矩陣的反矩陣也是同類型的基本矩陣）。

💡 洞察 高斯消去法的每一個步驟，都對應一個基本矩陣。LU 分解 = 把這些步驟「封裝」成 L 和 U 兩個矩陣。以後要解多次 Ax = b（不同 b），只要做一次 LU 分解，之後每次只要 O(n²)。
§2.5 馬可夫鏈（Markov Chains）

什麼是馬可夫鏈

一種描述「系統狀態隨機轉移」的數學模型。未來只與現在有關，與過去無關——這叫「馬可夫性質」。

x₍ₖ₊₁₎ = P · x₍ₖ₎

x₍ₖ₎ 是第 k 步時系統處於各狀態的機率向量，P 是轉移矩陣。

📊 轉移矩陣 P

n×n 隨機矩陣

  • 每個元素 Pᵢⱼ ≥ 0（機率非負）
  • 每列總和 = 1（機率守恆）
| 0.7 0.2 0.1 | | 0.1 0.8 0.1 | | 0.0 0.3 0.7 |

🔮 長期預測：Pᵏ

k步轉移機率

Pᵏ 的第 (i,j) 元素 = 從狀態 i 經過 k 步轉移到狀態 j 的機率。

當 k → ∞，系統收斂 → 得到「穩定態」（steady state）

⚡ 電網應用

機組狀態模擬

機組狀態（運轉/待機/維修）之間的轉移可以用馬可夫鏈建模。用來預測長期可用發電容量。

馬可夫鏈的長期行為

如果 P 是「正」的轉移矩陣（所有元素 > 0），則馬可夫鏈有唯一的穩定態：

x* = P · x*

即 x* 是 P 的特徵向量（特徵值 = 1）。這連接到 Ch7 的特徵值概念。

§2.6 矩陣運算的更多應用

🔗 圖與網絡的矩陣表示

鄰接矩陣（Adjacency Matrix）

N個節點的圖 → N×N 鄰接矩陣 A
Aᵢⱼ = 1 表示節點 i 和 j 有連接。

A² 的第 (i,j) 元素 = 從 i 到 j 的「長度為2」路徑數量
💬 電網應用 電網拓撲 = 圖。節點是 bus，邊是輸電線。鄰接矩陣告訴你哪些 bus 直接相連。

🖼️ 影像表示

影像 = 矩陣

灰階影像：一個像素值矩陣。
彩色影像：三個通道（RGB）各一個矩陣。

影像處理：模糊、銳化、邊緣檢測 → 都是特定的矩陣運算（卷積核）

📈 線性遞推關係

費波那契數列

| Fₖ₊₁ | | 1 1 | · | Fₖ | | Fₖ | = | 1 0 | | Fₖ₋₁ |

所以 [Fₙ₊₁, Fₙ]ᵀ = Aⁿ · [F₁, F₀]ᵀ

用矩陣冪次可以在 O(log n) 時間內算出第 n 個費波那契數！

Matrix
矩陣

Matrix Multiplication
矩陣乘法

Identity Matrix
單位矩陣

Inverse Matrix
反矩陣
A⁻¹ 滿足 A·A⁻¹ = A⁻¹·A = I。只有當 det(A) ≠ 0 時存在。
Elementary Matrix
基本矩陣
I 經過一次基本列運算後的矩陣。E·A = 對 A 做對應的列運算。
Transpose
轉置矩陣
Aᵀ：行列交換。(Aᵀ)ᵢⱼ = Aⱼᵢ。
Symmetric Matrix
對稱矩陣
A = Aᵀ。即 Aᵢⱼ = Aⱼᵢ。
