Ch1. Vector & Matrix Foundation ｜向量與矩陣基礎

線性代數的基石。所有資料、神經網路、電網感測器數據 — 都是向量。矩陣是變換和約束的語言。

章節狀態當前進度
核心關鍵字vector · matrix · rank · linear independence

📍 核心概念

📍 Vector | 向量

n維空間中的一個點或方向

有序的數字列表。每個數字是一個座標軸上的位置。電網中：一個bus的 [電壓, 相位, 頻率] 就是一個3維向量。

v = [3, 4] ∈ R² → 長度 ||v|| = √(3²+4²) = 5
💬 直覺 像GPS座標：[經度, 緯度, 海拔]。三個數字，但你已經精確知道位置了。向量就是「帶方向的數量」。
Rⁿ向量的加法純量乘法單位向量

🔢 Linear Combination | 線性組合

用係數乘以向量後相加

v = a₁u₁ + a₂u₂ + ... + aₙuₙ，其中 aᵢ 是純量（係數）。所有向量的線性組合構成一個集合，稱為 span(u₁, u₂, ...)。

v = 2·[1,0] + 3·[0,1] = [2, 3]
💬 直覺 就像用食材做菜：不同的基本向量是「原料」，純量係數是「用量」，線性組合就是「配方」。
span純量係數線性組合

🔄 Matrix | 矩陣

m列 × n行 的數字網格

矩陣 A(m×n) 把 n維向量 transform 成 m維向量。每一個線性變換都對應一個矩陣。神經網路一層：output = activation(W·input + b)。

A(m×n) · x(n×1) = b(m×1)
💬 直覺 像一個「配方」：吃進食材向量，按照矩陣規則攪拌烹飪，輸出成新的一道菜（新的向量）。
矩陣乘法單位矩陣 I轉置 Aᵀ逆矩陣 A⁻¹

📏 Linear Independence | 線性獨立

沒有一個向量能由其他向量表示

a₁v₁ + ... + aₙvₙ = 0 只有在所有 aᵢ = 0 時才成立。n個 n維線性獨立向量 → 可以張成整個 Rⁿ（就像建立了座標軸）。

[1,0], [0,1] 線性獨立｜[1,0], [0,1], [1,1] 不獨立（第三個 = 前兩個相加）
💬 直覺 就像「獨立的情報維度」。北和東是獨立的——往北走永遠不會讓你往東。但「北」和「北偏西30度」就不獨立了。
線性相關線性獨立極大線性無關組維度

🎯 Basis & Dimension | 基底與維度

向量空間的「最小生成器」

基底 = 一組線性獨立的向量，線性組合可以產生空間中所有向量。維度 = 基底中向量的個數。R³的標準基底：e₁=[1,0,0], e₂=[0,1,0], e₃=[0,0,1]。

任何 R³ 向量 [x,y,z] = x·e₁ + y·e₂ + z·e₃
💬 直覺 像建立座標系：你選了3個「測量方向」（不一定垂直），這3個方向加起來可以測量宇宙中任何位置。
標準基底維度生成空間座標表示

🔍 Rank | 秩

矩陣「真正獨立」的方向數

rank(A) = column space 的維度 = row space 的維度。告訴你這個矩陣「有多少真正不同的資訊方向」。full rank = 沒有冗餘資訊。

A(3×4), rank(A) ≤ min(3,4) = 3｜rank=3 → full row rank
💬 直覺 就像報告的「實質內容」vs「總頁數」。300頁如果只有3頁有意義的內容，rank = 3，不是300。
column rankrow rankfull rankrank deficiency

Ch2. Gaussian Elimination ｜高斯消去法

解線性方程組的系統化方法。電網的牛頓-拉夫森法（Newton-Raphson）每一次迭代，本質上都是在做高斯消去。

核心技能解 Ax = b · 求 rank · 求逆矩陣

Gaussian Elimination ｜高斯消去法

也稱：Row Reduction · 前向消去（Forward Elimination）

將增廣矩陣 [A|b] 通過列操作化為階梯形式（Row Echelon Form），然後用回代（Back Substitution）求出解。

⚙️ 步驟

  1. 写出增广矩阵 [A | b] — 把方程組右邊的常數放進矩陣最後一行
  2. 選第一列第一個非零元素為主元（pivot）
  3. 用列操作把 pivot 下面的元素變成 0（消去）
  4. 對第二列、第三列...重複以上步驟，直到矩陣變成上三角形
  5. 從最後一行開始回代，求出每個未知數
📐 範例：解方程組 x + y = 5
2x + 3y = 14

增廣矩陣：[1 1 | 5]
           [2 3 | 14]

R₂ → R₂ - 2·R₁：
[1 1 | 5]
[0 1 | 4]

回代：y = 4, x = 1 ✔️
pivot 主元row echelon form 階梯形式回代 back substitution列操作 row operations
💡 時間複雜度：O(n³) — n 是矩陣維度

🔑 三種情況判定

✅ 唯一解

滿足：rank(A) = rank([A|b]) = n（未知數個數）

增廣矩陣化簡後沒有自由列（free row）。每個未知數都能直接求出。

x + y = 5, x - y = 1 → x=3, y=2

♾️ 無限多解

滿足：rank(A) = rank([A|b]) < n

有自由變數（free variables）。解的個數 = 無窮多個，用參數表示。

x + y = 5 → x = 5 - t, y = t (t ∈ R)

✗ 無解

滿足：rank(A) < rank([A|b])

出現矛盾的行（如 [0 0 | 5]）。方程組不相容。

x + y = 5, x + y = 6 → 矛盾！

📖 詞彙

Augmented Matrix
增廣矩陣
[A|b] 把係數矩陣 A 和常數向量 b 合併成一個矩陣。
Pivot
主元
每列第一個非零元素。用來消去該列以下的所有元素。
Row Echelon Form (REF)
列階梯形式
每列第一個非零元素所在的行位置越往右下方。每列第一個非零元素右側全是零。
Reduced Row Echelon Form (RREF)
簡化列階梯形式
REF + 主元所在行其他元素皆為零。最簡形式，可直接讀出解。

Ch3. LU Decomposition ｜LU 分解

將矩陣 A 分解成 L（下三角矩陣）和 U（上三角矩陣）。高斯消去的矩陣視角封裝。LU 分解是實務上解線性方程組最快的工具之一。

核心技能A = L·U · A = L·D·U · PA = LU
應用解多次 Ax=b · 計算行列式 · 矩陣反矩陣

LU Decomposition ｜LU 分解

A = L·U（Lower · Upper）

把一個方陣 A 分解成一個下三角矩陣 L（Lower）和一個上三角矩陣 U（Upper）的乘積。如果存在，這種分解可以讓我們快速求解線性方程組。

A = L · U

⚙️ 核心概念

  1. U（Upper）= 高斯消去後得到的階梯形矩陣
  2. L（Lower）= 記錄你做了哪些列操作的「歷史」矩陣（對角線為1）
  3. 求 A⁻¹：解 n 次 Ly = eᵢ 再解 Ux = yᵢ（共 2n 次回代）
  4. 求 det(A)：det(A) = det(U) = 對角線元素乘積（避開 O(n³) 重新計算）
📐 範例 A = [2 1]
        [8 5]

高斯消去：R₂ → R₂ - 4·R₁
U = [2 1]
    [0 1]

L = [1 0]
     [4 1]

驗證：L·U = [1 0]·[2 1] = [2 1]
             [4 1] [0 1] [8 5] ✔️
L 下三角矩陣U 上三角矩陣pivot elements 主元Doolittle's algorithm
💡 為什麼重要：求一次 LU，之後每次解 Ax=b 只要 O(n²)。適合要解很多次同一個 A 但不同 b 的情況。

🔗 延伸：PA = LU（有行交換的情況）

⚠️ 為什麼需要 PA = LU？

當 pivot 位置是 0 時，需要交換列（row exchange）

如果某個 pivot 是 0（或接近 0，數值不穩定），就要交換列。這時用置換矩陣 P 記錄交換：A = Pᵀ·L·U 或 PA = LU。

PA = LU → L 和 U 仍然是三角矩陣
💬 直覺 就像排障礙物：如果第一個障礙物太高（=0），就先跳過，用下一個。P矩陣記錄了你「跳過了幾次」。

🔑 LU vs 其他分解

📊 A = L·D·U

L：單位下三角 · D：對角 · U：單位上三角

把 U 的對角元素抽出來放到 D，變成 L·D·U 形式。有時教學上更直觀。

📐 Cholesky 分解

A = Rᵀ·R（R 為上三角）

專門給對稱正定矩陣（symmetric positive definite matrix）。比 LU 快2倍。電網潮流計算常用。

📖 詞彙

Lower Triangular Matrix
下三角矩陣
對角線以上元素皆為0的矩陣。L 的對角線通常設為1。
Upper Triangular Matrix
上三角矩陣
對角線以下元素皆為0的矩陣。高斯消去後的 U 就是這種形式。
Permutation Matrix
置換矩陣
每行/每列只有一個1，其他都是0。PA = LU 中的 P 用來記錄列交換。
Cholesky Decomposition
Cholesky 分解
A = Rᵀ·R，專用於對稱正定矩陣。計算量是 LU 的一半。

Ch4. Inner Product Space ｜內積空間

在向量空間的基礎上，加上「長度、角度、距離」的概念。內積空間讓向量不只是一串數字，而是有了幾何意義。是你目前在課堂學習的部分。

章節狀態🎯 課堂進行中
核心技能內積 · 範數 · Cauchy-Schwarz · 夾角

💡 核心概念

· Inner Product | 內積

兩個向量之間的「乘法」

在 Rⁿ 中：u·v = u₁v₁ + u₂v₂ + ... + uₙvₙ。內積是衡量兩個向量「有多相似」的工具。

u = [1,2,3], v = [4,5,6] → u·v = 1·4 + 2·5 + 3·6 = 32
💬 直覺 就像「方向相近的兩個向量，內積越大」。完全相同方向 → 內積為正值。完全相反 → 負值。垂直 → 內積為0。
dot productu·v = Σuᵢvᵢ內積運算

📏 Norm | 範數（長度）

向量的「長度」

L² 範數（最常見）：||v|| = √(v·v) = √(v₁² + v₂² + ... + vₙ²)。這就是歐氏距離。

||[3,4]|| = √(3²+4²) = √25 = 5
💬 直覺 就是國中學的畢氏定理推廣到 n 維。從原點到點 (3,4) 的直線距離是5。
L² norm歐氏距離||v||unit vector

📐 Angle | 夾角

兩個向量之間的「方向差」

由內積導出：cos θ = (u·v) / (||u||·||v||)。範圍：-1 ≤ cos θ ≤ 1。

u=[1,0], v=[1,1] → cos θ = 1/√2 → θ = 45°
💬 直覺 夾角為0° = 完全同向 → cos=1。夾角90° = 垂直 → cos=0。夾角180° = 完全反向 → cos=-1。
cosine similaritycos θ夾角公式

📍 Distance | 距離

兩個向量的「差」的範數

d(u,v) = ||u - v|| = √((u₁-v₁)² + ... + (uₙ-vₙ)²)。這是歐氏距離。

u=[1,2], v=[4,6] → d = √((1-4)² + (2-6)²) = √(9+16) = 5
💬 直覺 就像地圖上兩點的直線距離。只問「有多遠」，不問「往哪裡」。
歐氏距離||u-v||L² distance

🔑 Cauchy-Schwarz Inequality

內積與範數之間的基本不等式

|u·v| ≤ ||u|| · ||v||。當且僅當 u 和 v 線性相關時取等號。這是一個「不可能超越的上限」。

|u·v| ≤ ||u||·||v|| → |cos θ| ≤ 1
💬 直覺 就像「不管怎麼選兩個向量，它們的夾角餘弦值永遠在 -1 到 1 之間」。這個看起來顯而易見的結論，數學上需要證明。
|u·v| ≤ ||u||·||v||基本不等式三角不等式

➕ Inner Product Properties

內積的四個基本性質

1. 對稱性：u·v = v·u
2. 正定性：v·v ≥ 0，且 v·v = 0 ⟺ v = 0
3. 線性性：(u+v)·w = u·w + v·w
4. 齊性：(cu)·v = c(u·v)

💬 直覺 這些性質保證了內積運算的「合理性和一致性」，就像加法和乘法滿足交換律、分配律一樣自然。

📖 詞彙

Inner Product
內積
兩個向量之間的運算，結果是純量。Rⁿ中即 dot product：u·v = Σuᵢvᵢ。
Norm (Length)
範數（長度）
||v|| = √(v·v)。向量的「大小」。滿足：||v||≥0，||cv||=|c|·||v||。
Unit Vector
單位向量
||v|| = 1 的向量。任何非零向量除以它的範數就是單位向量：v/||v||。
Cauchy-Schwarz Inequality
柯西-施瓦茲不等式
|u·v| ≤ ||u||·||v||。推論：|cos θ| ≤ 1。

Ch5. Orthogonality & Projection ｜正交性與投影

正交 = 垂直 = 內積為零。投影 = 找最近點。Gram-Schmidt = 把任意基底變成正交基底。最小平方法 = 用投影做曲線擬合。

核心技能Gram-Schmidt · 投影矩陣 · 最小平方法
應用回歸分析 · 訊號處理 · QR 分解

💡 核心概念

⊥ Orthogonal | 正交

u ⟂ v 當且僅當 u·v = 0

就像「北和東」——完全不重疊，往北走不會幫你往東走。正交的資訊通道之間沒有冗餘。

[1,0]·[0,1] = 0 → [1,0] ⟂ [0,1]
💬 直覺 正交 = 垂直。兩個正交的方向，測量其中一個完全不會影響另一個的讀數。

🎯 Orthogonal Projection | 正交投影

把向量投射到子空間上的「最近點」

projᵥ(u) = (u·v)/(v·v) · v。u 對 v 的投影，是 v 方向上最接近 u 的點。

projᵥ(u) = (u·v / v·v) · v
💬 直覺 像燈光打在牆上——你的影子（投影）落在牆面上。影子是你在牆面方向上「最近的你」。

🔧 Gram-Schmidt Process

把任意線性獨立向量組 → 正交基底

一步一步把每個向量減去它在前面向量上的投影，剩下的部分自然就和前面全部正交了。

v₁' = u₁｜v₂' = u₂ - projᵥ₁(u₂)｜v₃' = u₃ - projᵥ₁(u₃) - projᵥ₂(u₃)
💬 直覺 就像在房間裡豎起第一根柱子（v₁'），再立第二根時，先把第一根投影過去、減掉，確保新柱子不和舊柱子重疊。

📊 Least Squares | 最小平方法

找不到精確解時，求「誤差最小」的近似解

min ||Ax - b||²，等價於解 AᵀA x̂ = Aᵀb（正規方程）。x̂ 是在 column space 上的投影。

x̂ = (AᵀA)⁻¹Aᵀb — 當 AᵀA 可逆時
💬 直覺 就像用尺畫線通過一堆散點——不是所有點都在線上，但讓所有點到線的「垂直距離總和最小」。這就是回歸分析的本質。

📐 Projection Matrix | 投影矩陣

P = A(AᵀA)⁻¹Aᵀ — 把向量投影到 A 的 column space

投影矩陣的性質：P² = P（冪等），P = Pᵀ（對稱）。

Pb = 投影到 A 的 column space
💬 直覺 就像一個「投影燈」。不論你投影多少次（應用 P 兩次），結果都不會改變——投影後的東西再投影一次，還是它自己。

📏 QR Decomposition | QR 分解

Gram-Schmidt 的矩陣形式封裝

A = QR，Q 是 orthogonal（QᵀQ = I），R 是上三角。Q 的 columns 就是 A columns 經 Gram-Schmidt 正交化後的結果。

A = Q · R
💬 直覺 Q = 旋轉後的座標軸（R³的三個正交方向）。R = 在這些方向上的「座標值」。分解把複雜的 A 拆成「方向」和「強度」。

📖 詞彙

Orthogonal
正交
u·v = 0 意味著 u 與 v 垂直。正交向量是線性獨立的。
Orthonormal
正交規範（單位正交）
兩兩正交且每個向量都是單位向量。QᵀQ = I。
Gram-Schmidt
Gram-Schmidt 正交化
把線性獨立向量組轉換為正交（或正規）基底。Q = orth(A)。
Least Squares
最小平方法
min ||Ax - b||² 的解 x̂ = (AᵀA)⁻¹Aᵀb（當 AᵀA 可逆）。
Projection Matrix
投影矩陣
P = A(AᵀA)⁻¹Aᵀ。P² = P, P = Pᵀ。
QR Decomposition
QR 分解
A = QR，Q 正交，R 上三角。等價於 Gram-Schmidt 應用於 A。

Ch6. Eigenvalues & Eigenvectors ｜特徵值與特徵向量

Ax = λx — 矩陣作用的「本質」。特徵向量是變換後方向不變的向量，特徵值是縮放倍率。PCA、電網穩定性分析、PageRank — 全靠這個。

核心技能特徵方程式 · 對角化 · 幾何重數
應用電網穩定性 · PCA · 微分方程 · PageRank

💡 核心概念

🎯 Eigenvalue & Eigenvector

Ax = λx — 方向不變的向量

特徵向量 x 被矩陣 A 變換後，方向不變，只是縮放了 λ 倍。λ = 特徵值。

A = [2 1] → A·[1,0] = [2,0] = 2·[1,0] → λ=2, x=[1,0]
💬 直覺 就像轉動一個啞鈴——啞鈴的兩個把手（軸）方向不變，變的只是轉了多少度。軸 = eigenvector，轉角 = eigenvalue。
Ax = λxcharacteristic equationeigenvalue λeigenvector x

🔢 Characteristic Equation | 特徵方程式

det(A - λI) = 0

用來找 λ：把 λ 當成未知數，解這個多項式。λ 的根就是特徵值。

det([2-λ 1] - [0 0]) = (2-λ)(1-λ) - 0 = 0 → λ=2, λ=1 [3 2] [0 1-λ]
💬 直覺 就像問「這次線性變換，哪些方向是不會被旋轉的？」答案就是特徵向量。

🔗 Diagonalization | 對角化

A = SΛS⁻¹ — 換到特徵向量基底

如果 A 有 n 個線性獨立的特徵向量，就能對角化。在新座標系（特徵向量基底）下，A 只是一個「縮放器」Λ。

S = [eigenvectors] · Λ = diag(λ₁, λ₂, ..., λₙ)
💬 直覺 就像把房間的座標軸從「牆角」旋轉到「特徵向量方向」。在新座標下，複雜的 A 變成了「沿每個軸縮放 λᵢ 倍」的簡單操作。

↔️ Symmetric Matrix | 對稱矩陣

A = Aᵀ → 所有特徵值都是實數

對稱矩陣的特徵向量不僅存在，而且是兩兩正交的：A = QΛQᵀ（正交對角化）。

對稱矩陣：A = QΛQᵀ（Q 正交矩陣，Qᵀ = Q⁻¹）
💬 直覺 對稱矩陣就像「均勻的拉伸/壓縮」——沿著每個正交方向獨立地縮放。真實世界的很多物理系統都是對稱的。

🔄 電網穩定性：Eigenvalues 的應用

小信號穩定性分析

電力系統動態方程線性化後：Δẋ = J·Δx。Jacobian 矩陣 J 的 eigenvalues 決定系統穩定與否。

Re(λᵢ) < 0 → 振盪衰減（穩定）｜Re(λᵢ) > 0 → 振盪放大（失穩）
💬 直覺 把電網想成一個彈簧系統。如果輕輕推一下，系統搖晃後回到原位（λ 負）→ 穩定。如果推一下，晃得越來越大（λ 正）→ 不穩定，會斷電。

📊 Trace & Determinant

和與積：所有特徵值的代數關係

trace(A) = Σλᵢ = 對角線元素之和
det(A) = Πλᵢ = 特徵值乘積

A = [2 1] → trace = 3, det = 2 → λ₁+λ₂=3, λ₁λ₂=2 → λ=2,1 [0 1]
💬 直覺 trace = 戰略縱深（總戰力）。det = 佔領領土（最終成就）。兩個一起告訴你這場戰爭的所有重要資訊。

Ch7. SVD & PCA ｜奇異值分解與主成分分析

SVD 是所有矩陣分解的「通用貨幣」——任何矩陣都能分解。PCA = SVD 的應用。這是你的電網異常偵測研究的核心工具。

核心技能A = UΣVᵀ · SVD · PCA · 奇異值
應用異常偵測 · 降維 · 推薦系統 · 影像壓縮

💡 核心概念

🔍 SVD — Singular Value Decomposition

A = UΣVᵀ — 任何矩陣都適用的分解

U 和 V 是正交矩陣（UᵀU = I, VᵀV = I），Σ 是對角矩陣，對角線上是奇異值 σ₁ ≥ σ₂ ≥ ... ≥ 0。

A(m×n) = U(m×m) · Σ(m×n) · Vᵀ(n×n)
💬 直覺 把矩陣想成一個「訊號處理器」：Vᵀ 先旋轉 → Σ 沿每個方向縮放（奇異值=強度）→ U 再旋轉輸出。
σ₁ ≥ σ₂ ≥ ... ≥ 0left singular vectorsright singular vectors奇異值

📊 PCA — Principal Component Analysis

用 SVD 找資料中最重要的方向

數據矩陣 X → 中心化（減去均值）→ SVD(X) → V 的 columns = 主成分。取前 k 個 σ → 保留 95%+ 變異量。

PCA = SVD(X_centered) → 根據 σ 選擇主要成分 → 降維
💬 直覺 就像把一個3D形狀投影到2D平面，選擇那個「看起來最像原本形狀」的角度。第一主成分 = 投影後散點分佈最大的方向。

⚡ 電網異常偵測的應用

基於 SVD/PCA 的異常偵測流程

收集 N 個時間點的感測器數據 → 形成數據矩陣 X → SVD 分解 → 正常行為的主要成分 → 新的異常數據偏離主要成分 → 偵測！

重構誤差 = ||x - proj_{前k成分}(x)||² > 閾值 → 異常
💬 直覺 就像學習「正常的心電圖波形」——老師說正常長這樣。新的心電圖偏離這個模式太多 → 異常。

📐 Covariance Matrix | 共變異數矩陣

資料在各維度上的「共同變異」

Σ = (1/(n-1)) XᵀX（對中心化後的 X）。PCA = Σ 的特徵值分解。但 SVD 比特徵值分解更數值穩定。

Σ(i,j) = covariance of dimension i and j
💬 直覺 電網中：bus電壓和頻率可能共同變動（正相關）。共變異數矩陣告訴你所有維度兩兩之間「一起搖擺」的程度。

🔢 Pseudoinverse | 偽逆矩陣

A⁺ = VΣ⁺Uᵀ — 當 A 不可逆時的最佳近似

Σ⁺ 把 Σ 對角線上的非零 σ 換成 1/σ。當 A 可逆時，A⁺ = A⁻¹。最小平方法的解：x̂ = A⁺b。

x̂ = A⁺b — 最小範數最小平方解
💬 直覺 就像在方程無解時，找一個「最接近」的解。pseudo-inverse 就是這個「最接近」的代數表達。

📏 Low-Rank Approximation

用前 k 個奇異值近似原始矩陣

A ≈ U₁..ₖ Σ₁..ₖ V₁..ₖᵀ。誤差 = Σᵢ₌ₖ₊₁ σᵢ²。選擇 k 使得累計解釋變異量 ≥ 95%。

A_k = Σᵢ₌₁ᵏ σᵢ uᵢ vᵢᵀ
💬 直覺 把一本書的核心思想濃縮成3個要點——你忽略了細節，但捕捉到了最重要的模式。

📖 詞彙

Singular Values
奇異值
Σ 對角線上的非負實數。σ₁ ≥ σ₂ ≥ ... ≥ 0。奇異值的平方 = AᵀA 的特徵值。
Left/Right Singular Vectors
左/右奇異向量
U 的 columns = A 的左奇異向量。V 的 columns = A 的右奇異向量。U 和 V 都是正交的。
PCA
主成分分析
找資料變異量最大的正交方向。V 的 columns = 主成分。取前 k 個 → 降維。
Explained Variance
解釋變異量
σᵢ²/Σⱼσⱼ²。第 i 個奇異值解釋的變異量比例。取足夠的 k 讓累計 ≥ 95%。
Moore-Penrose Pseudoinverse
摩爾-潘洛斯偽逆
A⁺ = VΣ⁺Uᵀ。對於任意矩陣 A 都存在。當 A 可逆時等於 A⁻¹。
Reconstruction Error
重構誤差
||A - A_k||_F² = Σᵢ₌ₖ₊₁ σᵢ²。用前 k 個成分重構時遺漏的資訊量。
