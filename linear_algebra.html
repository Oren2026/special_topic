Ch1 線性方程式系統 ｜Linear Systems of Equations

面對多個未知數、多條約束時，如何系統化地找到解——或者判斷根本無解。兩種核心算法：高斯消去法（得到解）和高斯-喬登消去法（得到完整形式）。

線性方程式系統簡介 高斯消去法 & 高斯-喬登消去法 線性方程式系統的應用
§1.1 線性方程式系統簡介

什麼是「線性」方程式？

「線性」的意思是：未知數只會以「一次方」出現（沒有 x²、xy、sin(x) 這類東西），而且未知數之間不會相乘。

Linear（線性）的定義 每個方程式都是未知數的「線性組合 = 常數」的形式。
a₁x₁ + a₂x₂ + ... + aₙxₙ = b
其中 aᵢ 和 b 是已知常數，xᵢ 是未知數。

這看起來很限制，但它其實是大部分科學與工程問題的第一步近似——先搞清楚「線性的部分」，非線性的部分再慢慢處理。

方程式的標準形式

把 m 條方程式、n 個未知數寫在一起：

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
 ⋮                     ⋮
aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ

這就是「線性方程式系統」。有 m 條線性方程式，n 個未知數。我們的目標：找到所有滿足這些方程式的 (x₁, x₂, ..., xₙ)。

方程組的三種解的情況

情況一：唯一解
有且僅有一組 (x₁, x₂, ..., xₙ) 滿足所有方程式
條件：rank(A) = rank([A|b]) = n
情況二：無限多解
有无限多組解，解形成一個集合
條件：rank(A) = rank([A|b]) < n，有自由變數
情況三：無解
沒有任何解同時滿足所有方程式
條件：rank(A) < rank([A|b])，矛盾的行
§1.2 高斯消去法與高斯-喬登消去法

高斯消去法 ｜Gaussian Elimination

Forward Elimination + Back Substitution

將增廣矩陣 [A|b] 通過列運算化為上三角形（Row Echelon Form），然後從最後一行往前回代求出所有未知數。

⚙️ 標準步驟

  1. 寫出增廣矩陣 [A|b] — 把係數和常數寫在同一個矩陣
  2. 選第一列第一個非零元素為主元（pivot）
  3. 用列交換或列運算，把 pivot 下面的元素變成 0（前向消去）
  4. 對第二列、第三列... 重複，直到矩陣變成上三角形（台積階梯形）
  5. 從最後一行開始回代（Back Substitution），逐一求出每個未知數
📐 範例 x + y + z = 6
2x + 3y + z = 12
x + y + 2z = 7

增廣矩陣：
[1 1 1 | 6]
[2 3 1 | 12]
[1 1 2 | 7]

R₂→R₂-2R₁, R₃→R₃-R₁：
[1 1 1 | 6]
[0 1 -1 | 0]
[0 0 1 | 1]

回代：z=1, y=1, x=4 ✅
pivot 主元列交換列運算前向消去回代Row Echelon Form
💡 複雜度：O(n³) — 這是所有矩陣運算的基礎複雜度

高斯-喬登消去法 ｜Gauss-Jordan Elimination

完全化簡 → 直接讀出解

在高斯消去之後，繼續把每列的主元上方也消成 0，直到矩陣變成「簡化列階梯形」（Reduced Row Echelon Form）。這時可以直接讀出解——不需要回代。

⚙️ 與高斯消去的差異

  1. 做完高斯消去（上三角形）之後，不停下
  2. 繼續往上消——把每個 pivot 上方的元素也變成 0
  3. 最後結果：每個 pivot 是它所在列唯一的非零元素，且等於 1
  4. 好處：直接讀出 x 的值，不需要回代
📐 同一範例的 RREF [1 0 0 | 4] → x = 4
[0 1 0 | 1] → y = 1
[0 0 1 | 1] → z = 1

對照：高斯消去（只做到上三角形）需要 3 步回代，RREF 直接就是答案。
RREFReduced Row Echelon Form簡化列階梯形pivot = 1
💡 適合：用於求逆矩陣、解方程組、確認向量是否為解

高斯 vs 高斯-喬登 — 什麼時候用哪個？

高斯消去
只做到上三角形
計算量較少（比 RREF 快）· 適合只求方程組解
高斯-喬登
完全化簡到 RREF
計算量較大 · 適合求逆矩陣、確認向量是否為解、解多個同 A 的方程組

基本列運算（允許的操作）

在高斯/高斯-喬登消去中，你只能做三種「合法」操作：

  • 交換（Swap）： Ri ↔ Rj — 交換兩列。當 pivot 位置是 0 時使用。
  • 倍加（Scale）： Ri → c·Ri (c ≠ 0) — 把一列所有元素乘以非零常數。
  • 替換（Replace）： Ri → Ri + c·Rj — 把第 j 列的 c 倍加到第 i 列。這是最核心的操作。
💡 核心洞察 這三種操作不改變方程組的解集合。它們只是把方程組「換個樣子表示」，本質不變。
§1.3 線性方程式系統的應用

⚡ 電網節點分析（Node Analysis）

電路中的應用

利用克希荷夫電流定律（KCL）：流入節點的電流總和 = 流出節點的電流總和。N 個節點 → N-1 條線性方程式。

I₁ + I₂ + ... + Iₙ = 0 (at each node)
💬 比喻 就像自來水管網的節點——進入的水量必須等於離開的水量。所有節點的約束加起來，就是整個管網的流量分配。

📊 化學反應平衡

質量守恆定律

化學反應的反應物和生成物必須滿足原子守恆。配平化學方程式 = 解線性方程組。

CₓHᵧ + O₂ → CO₂ + H₂O
💬 比喻 就像在餐廳算帳——所有食材進來，必須等於所有菜品出去的質量。Carbon（碳）和Hydrogen（氫）必須balance。

🌐 網路流量規劃

流量守恆

交通網絡、電網、互聯網路由器——每個節點的流入流量等於流出流量。大型網絡 = 大型線性方程組。

💬 比喻 就像城市交通：進入每個路口的車流量，必須等於離開的車流量（假設沒有停車場）。整個城市交通的流量分配 = 解一個大方程組。

§2.1 矩陣運算

矩陣是什麼

一個 m×n 的矩陣是一個 m 列、n 行的數字長方形陣列。把線性方程組寫成矩陣形式：

A(m×n) · x(n×1) = b(m×1)

A 是 m×n 的係數矩陣，x 是 n×1 的未知數向量，b 是 m×1 的常數向量。

💡 幾何意涵 A(m×n) 是一個從 Rⁿ 空間到 Rᵐ 空間的「線性變換」。它把 n 維輸入向量 transform 成 m 維輸出向量。

矩陣的基本運算

➕ 矩陣加法

同維度的兩個矩陣，每個位置各自相加。

[1 2] + [5 6] = [6 8]
[3 4] [7 8] [10 12]

條件：A 和 B 必須同維度

✖️ 純量乘法

每個元素都乘以該純量 c。

3 · [1 2] = [3 6]
     [3 4] [9 12]

✖️ 矩陣乘法

A(m×n) · B(n×p) = C(m×p)。第 i 行第 j 列 = A第 i 行點乘 B第 j 列。

[1 2]·[5 7] = [1·5+2·9 1·7+2·10]
[3 4] [9 10] [3·5+4·9 3·7+4·10]
= [23 27]
  [51 61]

維度規則：中間維度必須相同，結果維度 = 外側維度

ᵀ 轉置（Transpose）

行列交換。第 (i,j) 元素變成第 (j,i) 元素。

[1 2 3]ᵀ = [1 4]
[4 5 6] [2 5]
          [3 6]

矩陣乘法的關鍵性質

⚠️ 矩陣乘法 ≠ 普通數字乘法
  • 交換律一般不成立： AB ≠ BA（通常不相等）
  • 零因子問題： AB = 0 不意味著 A=0 或 B=0
  • 消去律不成立： AB = AC 不意味著 B = C

但是結合律和分配律是成立的：

(AB)C = A(BC)  ✓ 結合律
A(B+C) = AB + AC  ✓ 分配律
(A+B)C = AC + BC  ✓ 分配律
§2.2 矩陣運算的性質

特殊矩陣類型

I 單位矩陣 I

對角線為1，其他位置為0。n×n。

I₃ = [1 0 0]
      [0 1 0]
      [0 0 1]

A·I = I·A = A（矩陣中的「1」）

D 對角矩陣

只在對角線有元素，其他為0。

D = diag(d₁, d₂, ..., dₙ)

對角矩陣的乘法很快：D·x = [d₁x₁, d₂x₂, ..., dₙxₙ]

T 上/下三角矩陣

上三角：對角線以下全是0。
下三角：對角線以上全是0。

上三角 = [★ ★ ★]
         [0 ★ ★]
         [0 0 ★]

三角矩陣的行列式 = 對角線元素乘積

S 對稱矩陣

A = Aᵀ。關於主對角線對稱。

[1 4 5]
[4 2 6]
[5 6 3]

真實世界的很多矩陣是對稱的（距離矩陣、共變異數矩陣）

O 零矩陣

所有元素為0。O + A = A。O·A = O。

P 置換矩陣

每行、每列恰好只有一個1，其餘為0。

用於「記錄列交換」：PA = LU 中的 P 就是置換矩陣。

矩陣的冪次

A² = A·A，A³ = A·A·A。當 A 是方陣（n×n）時有意義。

💡 實際應用 在馬可夫鏈中，Aᵏ 告訴你從狀態 i 經過 k 步轉移到狀態 j 的機率。
§2.3 反矩陣（Inverse Matrix）

什麼是反矩陣

對於方陣 A(n×n)，如果存在矩陣 B 使得 AB = BA = Iₙ，則 B 稱為 A 的反矩陣，記為 A⁻¹。

A·A⁻¹ = I   且   A⁻¹·A = I
類比 就像普通數字的倒數：5⁻¹ = 1/5，因為 5·(1/5) = 1。但矩陣的乘法沒有交換律，所以要滿足兩邊都是 I。
可逆（Invertible）
det(A) ≠ 0
又稱：nonsingular · nondegenerate
不可逆（Singular）
det(A) = 0
又稱：noninvertible · degenerate · singular

如何求反矩陣 — 高斯-喬登法

Augmented matrix trick

把 [A | I] 放在一起，用高斯-喬登消去把左邊變成 I，右邊剩下的就是 A⁻¹。

⚙️ 步驟

  1. 寫出增廣矩陣 [A | Iₙ]（左邊 A，右邊同維度單位矩陣）
  2. 對這個組合矩陣做高斯-喬登消去，把左邊變成 RREF
  3. 如果左邊變成 I，右邊就是 A⁻¹
  4. 如果左邊出現零列（rank不足）→ A 不可逆
📐 範例：求 [2 1] 的反矩陣 [1 1]

[A | I] = [2 1 | 1 0]
            [1 1 | 0 1]

R₁→R₁-R₂：[1 0 | 1 -1]
R₂→R₂-R₁：[0 1 | -1 2]

左邊 = I → A⁻¹ = [1 -1]
                    [-1 2]

驗證：A·A⁻¹ = [2 1]·[1 -1] = [1 0] = I ✅

反矩陣的重要性質

(A⁻¹)⁻¹ = A
(AB)⁻¹ = B⁻¹A⁻¹   （順序反過來！）
(cA)⁻¹ = (1/c)·A⁻¹  (c ≠ 0)
(Aᵀ)⁻¹ = (A⁻¹)ᵀ

反矩陣的應用：解線性方程組

當 A 可逆時，Ax = b 的唯一解可以直接寫成：

x = A⁻¹ · b
⚠️ 實際計算注意 雖然公式很漂亮，但直接算 A⁻¹ 需要 O(n³)。用高斯消去解 Ax = b 也需要 O(n³)，但常數更小。除非你要解很多個同樣的 A 但不同的 b，否則一般不推薦先算反矩陣再乘。
§2.4 基本矩陣（Elementary Matrices）

什麼是基本矩陣

單位矩陣 I 做一次「基本列運算」之後得到的矩陣。每一種基本列運算（交換、倍加、替換）都對應一個基本矩陣 E。

E·A = result of applying that row operation to A

也就是說：對 A 做一次列運算 = 在左邊乘上一個基本矩陣。

💬 比喻 基本矩陣就像「編輯指令」：I 是乾淨的空白文件，在上面做一次「刪除第二列」操作，就變成了「刪除矩陣」。把這個矩陣左乘到 A，等於對 A 執行同樣的「刪除第二列」操作。

↔️ 列交換型

I 的第 i 列和第 j 列交換後的結果。

× 列倍加型

I 的第 i 列乘以純量 c (c ≠ 0)。

+ 列替換型

I 的第 j 列加上第 i 列的 c 倍。

為什麼基本矩陣重要

高斯消去把 A 變成 U（上三角）的過程，相當於一系列基本矩陣的乘法：

Eₖ · ... · E₂ · E₁ · A = U

因此：

A = E₁⁻¹ · E₂⁻¹ · ... · Eₖ⁻¹ · U = L · U

其中 L = E₁⁻¹ · ... · Eₖ⁻¹ 是下三角矩陣（因為基本矩陣的反矩陣也是同類型的基本矩陣）。這就是 LU 分解的原理。

💡 洞察 高斯消去法的每一個步驟，都對應一個基本矩陣。LU 分解 = 把這些步驟「封裝」成 L 和 U 兩個矩陣。
§2.5 馬可夫鏈（Markov Chains）

什麼是馬可夫鏈

一種描述「系統狀態隨機轉移」的數學模型。每個狀態有固定機率轉移到其他狀態，未來只與現在有關（與過去無關）——這叫「馬可夫性質」。

x_{(k+1)} = P · x_{(k)}

x₍ₖ₎ 是第 k 步時系統處於各狀態的機率向量，P 是轉移矩陣（每個元素是機率）。

📊 轉移矩陣 P

n×n 隨機矩陣

每個元素 Pᵢⱼ ≥ 0（機率非負）
每列總和 = 1（機率守恆）

P = [0.7 0.2 0.1]
      [0.1 0.8 0.1]
      [0.0 0.3 0.7]

🔮 長期預測

Pᵏ 的意義

從狀態 i 經過 k 步轉移到狀態 j 的機率 = Pᵏ 的第 (i,j) 元素。

當 k → ∞，如果系統收斂 → 得到「穩定態」

⚡ 電網應用

發電廠狀態模擬

機組狀態（運轉/待機/維修）之間的轉移可以用馬可夫鏈建模。用來預測長期可用發電容量。

§2.6 矩陣運算的更多應用

🔗 圖與網絡的矩陣表示

鄰接矩陣（Adjacency Matrix）

N個節點的圖 → N×N 鄰接矩陣 A
Aᵢⱼ = 1 表示節點 i 和 j 有連接，否則為 0。

A² 的第 (i,j) 元素 = 從 i 到 j 的「長度為2」的路徑數量
💬 電網應用 電網拓撲 = 圖。節點是 bus，邊是輸電線。鄰接矩陣告訴你哪些 bus 直接相連。

🖼️ 影像表示

影像 = 矩陣

灰階影像：一個像素值矩陣。
彩色影像：三個通道（RGB）各一個矩陣。

影像處理：模糊、銳化、邊緣檢測 → 都是特定的矩陣運算（卷積核）

📈 線性遞推關係

費波那契數列

[Fₖ₊₁] = [1 1] · [Fₖ]
[Fₖ   ] [1 0] [Fₖ₋₁]

所以 [Fₙ₊₁, Fₙ]ᵀ = Aⁿ · [F₁, F₀]ᵀ。
用矩陣冪次可以在 O(log n) 時間內算出第 n 個費波那契數。

📖 詞彙（中英對照）

Matrix
矩陣
m×n 的數字長方形陣列。寫成 A(m×n) 表示 m 列 n 行。
Matrix Multiplication
矩陣乘法
A(m×n)·B(n×p) = C(m×p)。Cᵢⱼ = A的第i行 · B的第j列（點積）。
Identity Matrix
單位矩陣
Iₙ = diag(1,1,...,1)。A·I = I·A = A。矩陣中的「1」。
Inverse Matrix
反矩陣
A⁻¹ 滿足 A·A⁻¹ = A⁻¹·A = I。只有當 det(A) ≠ 0 時存在。
Elementary Matrix
基本矩陣
I 經過一次基本列運算後的矩陣。E·A = 對 A 做對應的列運算。
Transpose
轉置矩陣
Aᵀ：行列交換。(Aᵀ)ᵢⱼ = Aⱼᵢ。
Symmetric Matrix
對稱矩陣
A = Aᵀ。即 Aᵢⱼ = Aⱼᵢ。
Markov Chain
馬可夫鏈
隨機過程，未來只依賴現在狀態。x₍ₖ₊₁₎ = P·x₍ₖ₎，P 是轉移矩陣（隨機矩陣）。
