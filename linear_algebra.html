💡 Core Concepts — Plain English | 核心概念直解

📍 Vector | 向量

A list of numbers. Each number represents a coordinate along an axis. In the power grid: [voltage, phase_angle, frequency] at one instant.

v = [3, 4] → a point in 2D, or an arrow from origin to (3,4)
💬 直覺聯想 就像GPS座標：[經度, 緯度, 海拔]。只有3個數字，但你已經知道你在哪裡了。

🔄 Matrix | 矩陣

A rectangular grid of numbers. A matrix with m rows and n columns transforms n-dimensional vectors into m-dimensional vectors.

A(m×n) · x(n×1) = b(m×1)
💬 直覺聯想 像一個「配方」：吃進去食材（vector），按照矩陣的規則攪拌、烹飪，輸出成一道菜（新的vector）。

🎯 Eigenvalue & Eigenvector | 特徵值與特徵向量

Ax = λx — when you multiply eigenvector x by A, it only stretches/squishes (by λ), but direction doesn't change.

A·v = 3·v → v is eigenvector, λ=3 is eigenvalue
💬 直覺聯想 就像轉動一個物體時，旋轉軸本身的方向不變——軸就是eigenvector，轉動幅度就是eigenvalue。

🔍 Rank | 秩

How many independent directions (dimensions) a matrix actually has. A "full rank" matrix uses all its dimensions — no information is redundant.

rank(A) = number of pivot columns = dim(column space)
💬 直覺聯想 就像報告的「實質內容」vs 「總頁數」。300頁報告濃縮成3頁有意義的內容，rank = 3，不是300。

📐 SVD | 奇異值分解

A = UΣVᵀ — breaks any matrix into rotation (U), scaling (Σ), and rotation (Vᵀ). The singular values in Σ tell you how "important" each dimension is.

A = U · Σ · Vᵀ
💬 直覺聯想 像把一首交響樂拆成：樂器組成(U) → 各樂器音量(Σ) → 樂器排列(Vᵀ)。聽到音量就知道哪個樂器最重要。

📏 Orthogonal | 正交

Two vectors are orthogonal (perpendicular) when their dot product = 0. Orthogonal vectors are independent — they measure different things.

v · w = 0 → v ⟂ w
💬 直覺聯想 就像北和東——完全不重疊，往北走不會幫你往東走。兩個正交的資訊通道，不會重複。

🎯 Why This Matters for Your Research | 為什麼這對你的研究重要

📡 Anomaly Detection in Smart Grid

Collect N time steps of sensor readings → form a data matrix → apply PCA (which is just SVD on covariance). Anomalies = points far from the main data manifold.

🔗 Key LA: SVD · Eigenvalues · Covariance matrix · Dimensionality reduction

⚡ Power Flow & State Estimation

Power flow equations (nonlinear) → Newton-Raphson iteration → each step solves a linear system J·Δx = Δp. J = Jacobian matrix.

🔗 Key LA: Linear systems · LU decomposition · Jacobian · Eigenvalues (stability)

🧠 Neural Networks / Deep Learning

Forward pass = W·x + b repeated. Backpropagation = computing gradients of W via chain rule (partial derivatives of matrix operations). Loss landscape shaped by eigenvalues of Hessian.

🔗 Key LA: Matrix multiplication · Jacobian · Chain rule · Gradient descent

🔗 Graph Neural Networks (Grid Topology)

Power grid topology = graph. Adjacency matrix A encodes connections. Laplacian L = D - A. Eigenvalues of L reveal grid clusters and critical nodes.

🔗 Key LA: Adjacency matrix · Laplacian · Eigenvalues · Graph theory

📖 Core Vocabulary | 核心詞彙（中英對照）

Vector Space
向量空間
向量可以相加、可以被純量乘法後仍在空間內的集合。Rⁿ是最常見的例子。
Linear Independence
線性獨立
沒有一個向量可以表示成其他向量的線性組合。n個n維線性獨立向量可以張成整個Rⁿ空間。
Basis
基底
一組線性獨立的向量，線性組合可以產生空間中所有向量。R³的標準基底是[1,0,0], [0,1,0], [0,0,1]。
Linear Transformation
線性變換
保持向量加法和純量乘法的映射。每一個線性變換都對應一個矩陣。
Rank
秩
矩陣 column space 的維度。告訴你這個矩陣「真正有多少獨立方向」。
Null Space / Kernel
零空間 / 核
經過矩陣變換後變成零向量的所有向量。Null space告訴你哪些資訊在變換中消失了。
Determinant
行列式
det(A) = 變換的體積縮放因子。det=0表示奇異（不可逆）→ 資訊丟失。
Eigenvalue / Eigenvector
特徵值 / 特徵向量
Ax = λx。矩陣作用在特徵向量上，只會縮放不會改變方向。
Singular Value Decomposition (SVD)
奇異值分解
A = UΣVᵀ。適用於所有矩陣。Σ中的奇異值由大到小排列，前k個最大奇異值 captures 大部分資訊。
Orthogonal
正交
兩個向量的內積為零。它們代表「完全獨立的方向」。
Least Squares
最小平方法
找一個近似解讓誤差平方和最小。等價於在column space上做投影。
Projection
投影
將向量映射到子空間。回歸分析就是在找「最佳投影」。
Condition Number
條件數
最大與最小奇異值的比值。高的條件數 → 數值不穩定 → 解方程時誤差不斷放大。
Covariance Matrix
共變異數矩陣
資料在各維度上的相關性。PCA = 共變異數矩陣的特徵值分解。
