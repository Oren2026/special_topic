Ch1 線性方程式系統 ｜Linear Systems of Equations

面對多個未知數、多條約束時，如何系統化地找到解——或者判斷根本無解。兩種核心算法：高斯消去法（得到解）和高斯-喬登消去法（得到完整形式）。

線性方程式系統簡介 高斯消去法 & 高斯-喬登消去法 線性方程式系統的應用
§1.1 線性方程式系統簡介

什麼是「線性」方程式？

「線性」的意思是：未知數只會以「一次方」出現（沒有 x²、xy、sin(x) 這類東西），而且未知數之間不會相乘。

Linear（線性）的定義 每個方程式都是未知數的「線性組合 = 常數」的形式。
a₁x₁ + a₂x₂ + ... + aₙxₙ = b
其中 aᵢ 和 b 是已知常數，xᵢ 是未知數。

這看起來很限制，但它其實是大部分科學與工程問題的第一步近似——先搞清楚「線性的部分」，非線性的部分再慢慢處理。

方程組的標準形式

把 m 條方程式、n 個未知數寫在一起：

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁ a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂ ⋮ ⋮ aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ

這就是「線性方程式系統」。有 m 條方程式，n 個未知數。我們的目標：找到所有滿足這些方程式的 (x₁, x₂, ..., xₙ)。

方程組的三種解的情況

情況一：唯一解
有且僅有一組 (x₁, x₂, ..., xₙ) 滿足所有方程式
條件：rank(A) = rank([A|b]) = n
情況二：無限多解
有无限多組解，解形成一個集合
條件：rank(A) = rank([A|b]) < n，有自由變數
情況三：無解
沒有任何解同時滿足所有方程式
條件：rank(A) < rank([A|b])，矛盾的行

如何快速判斷解的情況

把增廣矩陣做高斯消去（只化到階梯形），觀察：

  • 化簡後，出現形如 [0 0 ... 0 | c]（c ≠ 0）的行 → 無解
  • 沒有矛盾行，且自由變數（free variable）個數 > 0 → 無限多解
  • 沒有矛盾行，且每個變數都是主元變數（pivot variable）→ 唯一解
§1.2 高斯消去法與高斯-喬登消去法

高斯消去法 ｜Gaussian Elimination

Forward Elimination + Back Substitution

將增廣矩陣 [A|b] 通過列運算化為上三角形（Row Echelon Form），然後從最後一行往前回代求出所有未知數。

⚙️ 標準步驟

  1. 寫出增廣矩陣 [A|b] — 把係數和常數寫在同一個矩陣
  2. 選第一列第一個非零元素為主元（pivot）
  3. 用列交換或列運算，把 pivot 下面的元素變成 0（前向消去）
  4. 對第二列、第三列... 重複，直到矩陣變成上三角形
  5. 從最後一行開始回代（Back Substitution），逐一求出每個未知數
pivot列交換列運算前向消去回代REF
💡 複雜度：O(n³) — 所有矩陣運算的基礎複雜度

高斯-喬登消去法 ｜Gauss-Jordan Elimination

完全化簡 → 直接讀出解

在高斯消去之後，繼續把每列的主元上方也消成 0，直到矩陣變成「簡化列階梯形」（RREF）。這時可以直接讀出解——不需要回代。

⚙️ 與高斯消去的差異

  1. 做完高斯消去（上三角形）之後，不停下
  2. 繼續往上消——把每個 pivot 上方的元素也變成 0
  3. 最後結果：每個 pivot 是它所在列唯一的非零元素，且等於 1
  4. 好處：直接讀出 x 的值，不需要回代
RREFpivot = 1簡化列階梯形不需要回代
💡 適合：用於求逆矩陣、解方程組、確認向量是否為解

高斯 vs 高斯-喬登 — 什麼時候用哪個？

方法做到什麼程度計算量適用場景
高斯消去 上三角形（REF） 較少 只求方程組的解
高斯-喬登 簡化列階梯形（RREF） 較多（多約50%） 求逆矩陣、確認解的狀態、多個右邊常數

基本列運算（允許的三種操作）

↔️ 交換（Swap）

Ri ↔ Rj

當 pivot 位置是 0 時使用。對應置換矩陣 P。

× 倍加（Scale）

Ri → c·Ri（c ≠ 0）

把一列所有元素乘以非零常數。

+ 替換（Replace）

Ri → Ri + c·Rj

最核心的操作。用於把元素消成 0。

💡 核心洞察 這三種操作不改變方程組的解集合。它們只是把方程組「換個樣子表示」，本質不變。
§1.3 線性方程式系統的應用

⚡ 電網節點分析（Node Analysis）

KCL：克希荷夫電流定律

流入節點的電流總和 = 流出節點的電流總和。N 個節點 → N-1 條線性方程式。

Σ Iᵢ = 0 (at each node)
💬 比喻 就像自來水管網的節點——進入的水量必須等於離開的水量。

📊 化學反應平衡

質量守恆定律

化學反應的反應物和生成物必須滿足原子守恆。配平化學方程式 = 解線性方程組。

💬 比喻 就像在餐廳算帳——所有食材進來，必須等於所有菜品出去的質量。

🌐 網路流量規劃

流量守恆

交通網絡、電網、互聯網路由器——每個節點的流入流量等於流出流量。

💬 比喻 就像城市交通：進入每個路口的車流量，必須等於離開的車流量。

📈 供需均衡模型

經濟學中的線性系統

N 種商品的市場供需均衡：每種商品的供給 = 需求，形成 N 條方程式。

💬 比喻 就像拍賣市場——當買家願付的價格等於賣家願收的價格時，市場成交。

Ch2 矩陣 ｜Matrix

矩陣不只是「記錄方程組」的容器——它本身是一個獨立的數學物件，有自己的代數規則。這章把「數字方塊」變成可以加減乘除的代數系統。

§2.1 矩陣運算

矩陣是什麼

一個 m×n 的矩陣是一個 m 列、n 行的數字長方形陣列。寫成 A(m×n) 表示 m 列、n 行。

A = | a₁₁ a₁₂ ... a₁ₙ | | a₂₁ a₂₂ ... a₂ₙ | | ⋮ ⋮ ⋱ ⋮ | | aₘ₁ aₘ₂ ... aₘₙ |

把線性方程組寫成矩陣形式：

A(m×n) · x(n×1) = b(m×1)

A 是係數矩陣，x 是未知數向量，b 是常數向量。

💡 幾何意涵 A(m×n) 是一個從 Rⁿ 空間到 Rᵐ 空間的「線性變換」。它把 n 維輸入向量 transform 成 m 維輸出向量。

矩陣的四種基本運算

➕ 矩陣加法

同維度的兩個矩陣，每個位置各自相加。

| 1 2 | | 5 6 | | 6 8 | | 3 4 | + | 7 8 | = | 10 12 |

條件：A 和 B 必須同維度

✖️ 純量乘法

每個元素都乘以該純量 c。

3 × | 1 2 | = | 3 6 | | 3 4 | | 9 12 |

✖️ 矩陣乘法

A(m×n) · B(n×p) = C(m×p)

C 的第 (i,j) 元素 = A 的第 i 行點乘 B 的第 j 列

ᵀ 轉置（Transpose）

行列交換。第 (i,j) 元素變成第 (j,i) 元素。

| 1 2 3 |ᵀ | 1 4 | | 4 5 6 | = | 2 5 | | 3 6 |

矩陣乘法的關鍵性質

⚠️ 矩陣乘法 ≠ 普通數字乘法
  • 交換律一般不成立： AB ≠ BA（通常不相等，甚至維度都不允許比較）
  • 零因子問題： AB = 0 不意味著 A=0 或 B=0
  • 消去律不成立： AB = AC 不意味著 B = C
✅ 成立的性質
  • 結合律：(AB)C = A(BC)
  • 左分配律：A(B+C) = AB + AC
  • 右分配律：(A+B)C = AC + BC

矩陣乘法的維度規則

A (m×n) · B (n×p) = C (m×p)

記住：中間維度（n）必須匹配，結果維度是「外側」的 m×p。

💡 記憶技巧 把維度豎著寫：A(m×**n**) · B(**n**×p) = C(m×p)。「吃進去的中間，吐出來的外側」。
§2.2 矩陣運算的性質

特殊矩陣類型

I 單位矩陣 I

對角線為1，其他位置為0。n×n。

I₃ = | 1 0 0 | | 0 1 0 | | 0 0 1 |

A·I = I·A = A（矩陣中的「1」）

D 對角矩陣

只在對角線有元素，其他為0。

D = diag(d₁, d₂, ..., dₙ)

D·x = [d₁x₁, d₂x₂, ..., dₙxₙ] — 乘法很快

↗️ 上/下三角矩陣

上三角：對角線以下全是0。
下三角：對角線以上全是0。

上三角 = | ★ ★ ★ | | 0 ★ ★ | | 0 0 ★ |

det(三角矩陣) = 對角線元素乘積

S 對稱矩陣

A = Aᵀ。關於主對角線對稱。

| 1 4 5 | | 4 2 6 | | 5 6 3 |

真實世界的很多矩陣是對稱的（距離矩陣、共變異數矩陣）

P 置換矩陣

每行、每列恰好只有一個1，其餘為0。

| 0 1 0 | | 1 0 0 | | 0 0 1 |

用於「記錄列交換」：PA = LU 中的 P

O 零矩陣

所有元素為0。

O + A = A · O·A = O · 但要注意：O·A = O 不意味著 O = 0 或 A = 0！

矩陣的冪次

A² = A·A，A³ = A·A·A。當 A 是方陣（n×n）時才有意義。

💡 實際應用 在馬可夫鏈中，Aᵏ 告訴你從狀態 i 經過 k 步轉移到狀態 j 的機率。
§2.3 反矩陣（Inverse Matrix）

什麼是反矩陣

對於方陣 A(n×n)，如果存在矩陣 B 使得 AB = BA = Iₙ，則 B 稱為 A 的反矩陣，記為 A⁻¹。

A · A⁻¹ = I 且 A⁻¹ · A = I
可逆（Invertible）
det(A) ≠ 0
又稱：nonsingular · nondegenerate
不可逆（Singular）
det(A) = 0
又稱：noninvertible · degenerate

如何求反矩陣 — 高斯-喬登法

Augmented matrix trick

把 [A | I] 放在一起，用高斯-喬登消去把左邊變成 I，右邊剩下的就是 A⁻¹。

⚙️ 步驟

  1. 寫出增廣矩陣 [A | Iₙ]（左邊 A，右邊同維度單位矩陣）
  2. 對這個組合矩陣做高斯-喬登消去，把左邊變成 RREF
  3. 如果左邊變成 I，右邊就是 A⁻¹
  4. 如果左邊出現零列（rank不足）→ A 不可逆
RREF[A|I]nonsingulardet(A) ≠ 0

反矩陣的重要性質

性質公式
反矩陣的反矩陣(A⁻¹)⁻¹ = A
乘積的反矩陣(AB)⁻¹ = B⁻¹A⁻¹（順序反過來！）
純量乘法(cA)⁻¹ = (1/c)·A⁻¹ (c ≠ 0)
轉置的反矩陣(Aᵀ)⁻¹ = (A⁻¹)ᵀ

反矩陣的應用：解線性方程組

當 A 可逆時，Ax = b 的唯一解可以直接寫成：

x = A⁻¹ · b
⚠️ 實際計算注意 直接算 A⁻¹ 需要 O(n³)，用高斯消去解 Ax = b 也需要 O(n³)，但常數更小。除非你要解很多個同樣的 A 但不同的 b，否則一般不推薦先算反矩陣再乘。
§2.4 基本矩陣（Elementary Matrices）

什麼是基本矩陣

單位矩陣 I 做一次「基本列運算」（交換、倍加、替換）之後得到的矩陣。

E · A = 對 A 進行對應列運算後的結果

也就是說：對 A 做一次列運算 = 在左邊乘上一個基本矩陣 E。

↔️ 列交換型

I 的第 i 列和第 j 列交換後的結果。反矩陣 = 自身（交換兩次回到原樣）。

× 列倍加型

I 的第 i 列乘以純量 c (c ≠ 0)。反矩陣：第 i 列乘以 1/c。

+ 列替換型

I 的第 j 列加上第 i 列的 c 倍。反矩陣：第 j 列減去第 i 列的 c 倍。

為什麼基本矩陣重要 — LU 分解的原理

高斯消去把 A 變成 U（上三角）的過程，相當於一系列基本矩陣的乘法：

Eₖ · ... · E₂ · E₁ · A = U

因此：

A = E₁⁻¹ · E₂⁻¹ · ... · Eₖ⁻¹ · U = L · U

L 是下三角矩陣（因為基本矩陣的反矩陣也是同類型的基本矩陣）。

💡 洞察 高斯消去法的每一個步驟，都對應一個基本矩陣。LU 分解 = 把這些步驟「封裝」成 L 和 U 兩個矩陣。以後要解多次 Ax = b（不同 b），只要做一次 LU 分解，之後每次只要 O(n²)。
§2.5 馬可夫鏈（Markov Chains）

什麼是馬可夫鏈

一種描述「系統狀態隨機轉移」的數學模型。未來只與現在有關，與過去無關——這叫「馬可夫性質」。

x₍ₖ₊₁₎ = P · x₍ₖ₎

x₍ₖ₎ 是第 k 步時系統處於各狀態的機率向量，P 是轉移矩陣。

📊 轉移矩陣 P

n×n 隨機矩陣

  • 每個元素 Pᵢⱼ ≥ 0（機率非負）
  • 每列總和 = 1（機率守恆）
| 0.7 0.2 0.1 | | 0.1 0.8 0.1 | | 0.0 0.3 0.7 |

🔮 長期預測：Pᵏ

k步轉移機率

Pᵏ 的第 (i,j) 元素 = 從狀態 i 經過 k 步轉移到狀態 j 的機率。

當 k → ∞，系統收斂 → 得到「穩定態」（steady state）

⚡ 電網應用

機組狀態模擬

機組狀態（運轉/待機/維修）之間的轉移可以用馬可夫鏈建模。用來預測長期可用發電容量。

馬可夫鏈的長期行為

如果 P 是「正」的轉移矩陣（所有元素 > 0），則馬可夫鏈有唯一的穩定態：

x* = P · x*

即 x* 是 P 的特徵向量（特徵值 = 1）。這連接到 Ch7 的特徵值概念。

Ch3 行列式 ｜Determinants

行列式是附著在「方陣」上的一個純量。它回答：一個矩陣是不是「好的」（可逆）？以及它代表的線性變換的面積/體積改變了多少？

矩陣的行列式 行列式與基本運算 · 行列式的性質 Cramer's Rule · 幾何意涵
§3.1 矩陣的行列式

行列式的定義：從 2×2 開始

行列式是只有「方陣」（n×n）才有的屬性。記作 det(A) 或 |A|。

2×2 基本公式

| a b | | c d | = ad − bc

記憶技巧：ad 減去 bc（反對角線減正對角線）

3×3 代數餘因子展開

沿第一行展開：

det(A) = a₁₁C₁₁ + a₁₂C₁₂ + a₁₃C₁₃ where Cᵢⱼ = (−1)ⁱ⁺ʲ · det(Mᵢⱼ)

Mᵢⱼ = 劃去第 i 列第 j 行後的 (n−1)×(n−1) 子矩陣

幾何意涵：行列式 = 面積（或體積）縮放因子

💡 幾何直覺 在二維：如果 A 是 R² 中的一個線性變換，則 |det(A)| = A 將「單位正方形」變換後的面積。
  • det(A) > 0 → 方向不翻轉（右手座標系保持右手座標系）
  • det(A) < 0 → 方向翻轉（鏡射）
  • det(A) = 0 → 面積塌縮成 0 → 維度降低 → 不可逆！
📌 為什麼這個重要（連結你的研究） 在電網異常偵測中，數據矩陣的「SVD 奇異值」等於行列式的推廣概念。當奇異值變得很小（接近 0），代表數據矩陣「快要不可逆」——這正是異常偵測的訊號。
§3.2 行列式與基本運算

三種基本列運算對行列式的影響

這個章節的價值：用列運算把複雜矩陣變簡單，再算行列式。

↔️ 列交換（Swap）

det(P) = −1

交換一次 → det 變號。交換兩次 → 回到原值。

× 列倍加（Scale）

det(c·Rᵢ) = c · det(A)

把一列乘以 c，行列式也乘以 c。

+ 列替換（Replace）

det(Rᵢ + c·Rⱼ) = det(A)

最神奇的性質：這個操作不改變 det。

實際應用：用列運算簡化行列式計算

把矩陣用列運算變成上三角形，然後直接取對角線乘積。

💡 實用算法 如果能用「列替換」把矩陣變成上三角形（不交換、不倍加），則：
上三角矩陣的 det = 對角線元素的乘積
這比直接用代數餘因子展開快很多——尤其矩陣變大的時候。
§3.3 行列式的性質

最重要的性質

性質公式重要程度
乘積的行列式det(AB) = det(A)·det(B)⭐⭐⭐⭐⭐

det(AB) = det(A)·det(B) 的深層意義

為什麼這個性質這麼重要？

如果 A 和 B 都是可逆的（det ≠ 0），則 AB 的可逆性由 det(A)·det(B) 決定。

推論：如果 det(A) ≠ 0，則 A 是一對一的映射（有反函數）。

⚠️ 常見誤解
  • det(A+B) ≠ det(A) + det(B) — 這個公式根本不存在！
  • det(A) = 0 ≠ A = 0 — 零矩陣行列式為零，但行列式為零的矩陣不一定是零矩陣

det(A) = 0 的完整意義（連結 Ch2）

det(A) ≠ 0
A 可逆（nonsingular）
rank(A) = n · A 的列全部線性獨立
det(A) = 0
A 不可逆（singular）
rank(A) < n · 列線性相關 · Ax=0 有非零解
§3.4 行列式的應用

Cramer's Rule：用行列式優雅地解方程組

對於 n×n 方程組 Ax = b（當 A 可逆時），Cramer's Rule 說：

xᵢ = det(Aᵢ) / det(A)

其中 Aᵢ 是把 A 的第 i 行置換成 b 向量後的矩陣。

⚠️ 數值警告 Cramer's Rule 看起來很漂亮，但計算複雜度是 O(n!) 或 O(n⁴)。實際上——高斯消去只需要 O(n³)。這個公式只適合「紙上推導」和「小型理論證明」，不適合實際計算。記住它很漂亮，但果斷放棄用它算大型方程組。

行列式的其他應用

📐 幾何：面積與體積

三個點 (x₁,y₁), (x₂,y₂), (x₃,y₃) 決定的三角形面積：

| x₁ y₁ 1 | | x₂ y₂ 1 | / 2 | x₃ y₃ 1 |

在電網中可用於電壓三角形、功率三角形的面積分析。

🧮 行列式測試

在演算法中，快速檢測矩陣是否接近奇異：

如果 det(A) ≈ 0，矩陣接近不可逆，數值計算時會遇到「病態條件」問題。

💬 連結你的研究 PCA 中，如果數據矩陣的行列式（所有奇異值的乘積）非常小，說明數據的「有效維度」遠低於原始維度——這就是維度壓縮的依據。

Ch4 向量空間 ｜Vector Spaces

這章是線性代數的骨幹。從具體的 Rⁿ 出發，慢慢推廣到「只要能線性組合」的東西都是向量——多項式、函數、聲音訊號。全都適用同一套理論。

Rⁿ的向量 · 向量空間定義 子空間 · 生成與獨立 · 基底與維度 · 秩 座標變換 · 電網/ML應用
§4.1 Rⁿ 的向量

Rⁿ 是什麼

Rⁿ 是所有 n 個實數組成有序 tuple 的集合。

R¹ 數線

一維：所有實數。可以用一個數字定位一個點。

R² 平面

二維：(x, y)。用兩個座標定位平面上一個點。

R³ 空間

三維：(x, y, z)。用三個座標定位空間中一個點。

Rⁿ 超空間

n 維：沒辦法畫圖，但數學上完全合法。每個維度可以是一個感測器、一個時間點...

向量的兩種視角

幾何視角
向量 = 帶方向和大小的箭頭
加法 = 頭接尾的移動 · 純量乘法 = 拉長/縮短/反向
代數視角
向量 = 有序 n 元組 (x₁, x₂, ..., xₙ)
加法 = 各分量相加 · 純量乘法 = 各分量乘以純量
兩種視角同等重要 幾何視角幫你建立直覺（R² 和 R³ 可以畫圖）。代數視角讓你處理任何維度的問題（N 個感測器 = Rⁿ）。你的碩論處理的電力數據：100 個時間點 × 50 個感測器 = R⁵⁰⁰⁰ 的向量。

向量的基本運算

➕ 向量加法

(1, 2, 3) + (4, 5, 6) = (5, 7, 9)

各分量分開加。

✖️ 純量乘法

3 · (1, 2, 4) = (3, 6, 12)

每個分量都乘以純量 3。

· 點積（Dot Product）

(1,2,3) · (4,5,6) = 1×4+2×5+3×6 = 32

對應分量相乘後相加。這個連接到 Ch5 的內積概念。

§4.2 向量空間（抽象定義）

為什麼要抽象化？

R²、R³ 很具體。但現實中，很多東西可以「線性組合」——

  • 多項式：a₀ + a₁x + a₂x²（向量空間 P₂）
  • 矩陣：m×n 矩陣可以相加、純量乘法（向量空間 Mₘₙ）
  • 聲音訊號：離散時間訊號序列（向量空間 Rᵀ）
  • 函數：連續函數（向量空間 C[a,b]）

如果我們只會處理 Rⁿ，那上面這些全部都要用不同的工具。但如果它們都滿足同一套公理，就可以用同一套理論處理。

向量空間的八條公理

一個集合 V 如果對「加法」和「純量乘法」封閉，且滿足以下八條公理，就是向量空間：

A₁-4 加法封閉性

  • A1：u + v = v + u（交換律）
  • A2：(u+v)+w = u+(v+w)（結合律）
  • A3：存在零向量 0 使得 v+0 = v
  • A4：每個 v 都有負向量 −v 使得 v+(−v) = 0

M₁-4 純量乘法封閉性

  • M1：1·v = v（單位元）
  • M2：(cd)·v = c·(d·v)（結合律）
  • M3：c·(u+v) = c·u + c·v（分配律）
  • M4：(c+d)·v = c·v + d·v（分配律）
💡 記住的技巧 A1-A4 = 「加法要有群結構」（交換群）。M1-M4 = 「純量乘法要有線性性質」。滿足這八條的，就是向量空間——跟你的具體集合是什麼無關。

向量空間的例子

向量空間元素長什麼樣維度
Rⁿ(x₁, x₂, ..., xₙ)n
Pₙa₀ + a₁x + ... + aₙxⁿn+1
Mₘₙm×n 矩陣m×n
Rᵀ(x₀, x₁, ..., xᵀ₋₁) 訊號序列T
C[a,b][a,b] 上的連續函數無限維
⚡ 連結你的研究 電網中的測量數據：1000 個時間點 × 50 個 bus = 50,000 維向量。在這個向量空間裡，「正常數據」構成一個低維子空間，異常是偏離這個子空間的向量。PCA 就是把這個低維子空間找出來的工具。
§4.3 向量空間的子空間

什麼是子空間

向量空間 V 的子集 H，如果對加法和純量乘法封閉，則 H 本身就是一個向量空間——稱為 V 的子空間。

封閉性檢驗（Subspace Test） 要證明 H 是 V 的子空間，只需要確認：
  • ① 零向量 0 在 H 中
  • ② 封閉加法：u, v ∈ H ⇒ u+v ∈ H
  • ③ 封閉純量乘法：v ∈ H, c ∈ R ⇒ c·v ∈ H

重要的子空間：核（Null Space）與像（Column Space）

ker(A) 核（零空間）

ker(A) = { x ∈ Rⁿ | Ax = 0 }

Ax = 0 的所有解的集合。
是 Rⁿ 的一個子空間。

幾何：穿過原點的直線或平面

im(A) 像（列空間）

im(A) = { Ax | x ∈ Rⁿ } ⊆ Rᵐ

A 的所有行向量線性組合的結果。
是 Rᵐ 的一個子空間。

幾何：Rᵐ 中的某個平面或線

為什麼核與像是最重要的子空間？

Dimension Theorem（維度定理）
dim(ker(A)) + dim(im(A)) = n（A 的列數）
這個定理把「解方程組」（找核）和「求值域」（成像）連結起來。當你知道其中一個維度，就能推出另一個。
⚡ 連結你的研究：電網狀態估計 測量值 y = Ax + 雜訊。x 是電網狀態（電壓相位）。A 是測量矩陣。
  • 如果 y 不在 im(A) 中 → 測量有異常或壞資料
  • y 在 im(A) 中的程度 → 決定了數據的「一致性」
這就是「不良資料偵測」（Bad Data Detection）的線性代數原理。
§4.4 生成集合與線性獨立

Span（張成）：我能用這些向量碰出多大的空間？

Span{v₁, v₂} = { c₁v₁ + c₂v₂ | c₁, c₂ ∈ R }

所有可能的線性組合。幾何上：是 v₁ 和 v₂ 決定的平面（如果它們線性獨立）。

線性獨立：這組向量裡有沒有「多餘的」？

定義：線性獨立 v₁, v₂, ..., vₖ 線性獨立 ⟺ c₁v₁ + c₂v₂ + ... + cₖvₖ = 0 只有 c₁=c₂=...=cₖ=0 這組解

翻譯：沒有任何一個向量可以表示成其他向量的線性組合。
線性獨立
v₁ = (1,0), v₂ = (0,1) 在 R² 中
c₁(1,0)+c₂(0,1)=(0,0) 只有 c₁=c₂=0
線性相關
v₁ = (1,0), v₂ = (2,0) 在 R² 中
2·v₁ − 1·v₂ = (0,0)，有非零解

幾何直覺：R³ 中的線性相關

1個向量 v ≠ 0

Span{v} = 穿過原點的一條直線

1 個維度

2個無關向量 v₁, v₂ 無關

Span{v₁, v₂} = 穿過原點的平面

2 個維度

3個無關向量 v₁, v₂, v₃ 無關

Span{v₁, v₂, v₃} = 整個 R³ 空間

3 個維度

3個相關向量 v₃ = v₁ + v₂

Span{v₁, v₂, v₃} = Span{v₁, v₂} = 那個平面（v₃ 沒貢獻新維度）

仍然是 2 個維度，v₃ 多餘

§4.5 基底與維度

基底：向量空間的「最小生成集」

基底的定義 B = {v₁, v₂, ..., vₙ} 是 V 的基底，當且僅當：
  • ① B 中的向量線性獨立（沒有多餘的）
  • ② Span(B) = V（張成整個空間）

維度：基底的向量個數

同一個向量空間的所有基底，向量個數相同。這個數字叫做維度。

標準基底 R³

{(1,0,0), (0,1,0), (0,0,1)}

三個向量，互相垂直，長度為 1。最「自然」的基底。

非標準基底 R²

{(1,1), (1,-1)}

這也是 R² 的基底！向量 (3,1) 在標準基底是 (3,1)，在這個基底是 (2,1)——同一個向量，不同座標。

⚡ 連結你的研究：PCA 就是找更好的基底 你的感測器數據本來用「每個感測器」做一個維度（標準基底）。但這個基底可能不是最好的——很多維度之間高度相關。PCA 找的是：找出「數據變異最大方向」的新基底，用少數幾個維度就能捕捉大部分資訊。
§4.6 矩陣的秩與線性方程式系統

rank(A) 的完整定義（連結 Ch1）

Ch1 的 rank = 方程組有多少 pivot。這只是表面。Ch4 的 rank = 向量空間的維度。

Ch1 的 rank（工具視角）
A 做高斯消去後，非零列的個數
用來判斷方程組解的情況
Ch4 的 rank（結構視角）
dim(col(A)) = dim(row(A))
行空間維度 = 列空間維度 = 不變的結構量

維度定理的完整陳述

dim(ker(A)) = n − rank(A) dim(im(A)) = rank(A)
這個定理的實際用法 當你用高斯消去把 A 變成 RREF 之後：
  • 非零列的個數 = rank(A)
  • 自由變數的個數 = n − rank(A) = dim(ker(A))
這把「算法」（高斯消去）和「結構」（維度）連結起來了。

rank 與方程組解的完整分類

條件幾何意義方程組解
rank(A) = rank([A|b]) = n列空間維度 = n唯一解
rank(A) = rank([A|b]) < nker(A) 有維度 > 0無限多解（自由度 = n−rank）
rank(A) < rank([A|b])b 不在 col(A) 中無解（矛盾）
§4.7 座標與基底變換

同一個向量，不同基底，不同座標

這是 Ch4 最實用的一節。學完這節，你就知道 PCA 為什麼 work。

標準基底 S = {(1,0), (0,1)} in R²

向量 v = (3,1) 在 S 中：座標 = (3,1)

意義：v = 3·e₁ + 1·e₂

新基底 B = {(1,1), (1,-1)}

同一個 v = (3,1) 在 B 中：座標 = (2,1)

意義：v = 2·(1,1) + 1·(1,-1)

基底變換矩陣（Transition Matrix）

從舊基底到新基底的轉換：

[v]ᵦ = P⁻¹ · [v]ₛ

P 的行 = 新基底的每個向量在舊基底中的座標

💡 PCA 的線性代數原理 PCA 找的新基底 = 數據協變異數矩陣的特徵向量組成的基底。
  • 原始數據：用標準基底 → 維度之間高度相關
  • PCA基底：用特徵向量基底 → 維度之間完全不相關（正交）
  • 保留前 k 個維度 → 維度壓縮，去除雜訊
這整個過程，就是「座標變換」。
§4.8 向量空間的應用

⚡ 電網狀態估計

測量空間 → 狀態空間

測量值 y ∈ Rᵐ 是 m 維，電網狀態 x ∈ Rⁿ 是 n 維（n ≪ m 通常）。

y = Ax + ε（測量模型）

A 的 rank 決定了「可觀測狀態」的數量。

💬 向量空間視角 「可觀測狀態」 = im(A)。如果 rank(A) = n，則所有狀態都可以從測量值重建（狀態估計可行）。

🧠 機器學習：特徵空間

所有的「特徵」構成一個向量空間

每個數據點 = 特徵空間中的一個向量。

維度 = 特徵數量。

PCA = 在特徵空間中找更好的基底。

📡 信號處理：時域 ↔ 頻域

傅立葉變換 = 基底變換

時域：訊號用 (1, t, t²...) 基底表示。

頻域：用 (sin(nt), cos(nt)) 基底表示。

同一個訊號，不同基底，資訊的呈現方式不同。

🎨 影像壓縮：JPEG

DCT = 另一種基底變換

原始影像：每個像素是一維，維度 = 像素數。

DCT基底：8×8 區塊用頻率基底表示。

壓縮：保留低頻係數（少數維度），丟棄高頻係數（視覺影響小的細節）。

📖 詞彙（中英對照）

Vector Space
向量空間
滿足八條公理的集合，對加法和純量乘法封閉。V = (V, +, ·)
Subspace
子空間
V 的子集 H，對加法和純量乘法封閉。包含零向量。
Span
張成空間
Span{v₁,...,vₖ} = 所有 c₁v₁+...+cₖvₖ 的集合。
Linear Independence
線性獨立
c₁v₁+...+cₖvₖ=0 只有零解 ⟺ 沒有向量可表示成其他向量的線性組合。
Basis
基底
線性獨立 + 張成整個空間。基底的向量個數 = 維度。
Dimension
維度
向量空間的基底中向量的個數。記作 dim(V)。
Null Space (Kernel)
核 / 零空間
ker(A) = {x | Ax = 0}。Ax=0 的所有解的集合。
Column Space (Image)
列空間 / 像
im(A) = {Ax | x ∈ Rⁿ}。A 的行向量能張成的空間。
Rank
秩
rank(A) = dim(col(A)) = dim(row(A))。也是非零 pivot 的個數。
Transition Matrix
基底變換矩陣
P 的行 = 新基底的向量在舊基底中的座標。換基底：P⁻¹x。
Change of Basis
座標變換
同一個向量在不同基底下的座標不同。PCA 就是換基底。
