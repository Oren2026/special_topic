🔴 角袋（4個）

位於球檯四個角落，上左/上右/下左/下右。洞口徑 50mm。

🟡 側袋（2個）

位於短邊（630mm）中央，上中/下中。洞口徑 50mm。長邊（1200mm）在上下側。

口袋內緣 = 洞口徑 50mm 內側邊緣（球心到此距離 < 25mm 視為進袋）

球心約束：球心座標必須在 felt 內部，不得接觸庫邊橡膠
進袋條件：球心距離口袋中心 < 25mm（洞口徑/2）
路徑約束：Ghost Ball 瞄準線不得穿越庫邊（銀行球除外）
擊球約束：Ghost Ball 與目標球必須分居口袋兩側

🎯 演算法需求（待討論）

  1. 口袋選擇：6個口袋誰來決定？手動點擊？或系統評分？
  2. 障礙判斷：路徑直線上有無障礙球（目標球兩側 19mm 緩衝）
  3. 庫邊反彈：直線行不通時，選擇哪條庫邊？鏡像法如何驗證反射角？
  4. 白球剝離：擊球後白球會滾向哪裡？停車位置如何估算？
  5. 9號球特殊：需最後打進，需避開直接路徑被封鎖的情形
